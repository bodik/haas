About HaaS cesnet.cz -- Honeypot as a Service suite

This software suite is designed to aid creating and maintaining data processing cloud within a realm of CESNET.CZ and beyond. It is based on masterless puppet and bash. Whole suite is bootsrapped from git to the clean provisioned Debian Stretch VM.
